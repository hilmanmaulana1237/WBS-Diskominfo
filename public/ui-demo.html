📖 Informasi Enhancement

Fitur Otomatis
  • Ripple effect pada semua button
  • Smooth scroll untuk anchor links
  • Navbar scroll effect
  • Scroll progress bar di top
  • Counter animation untuk stat cards
  • Table hover dengan gradient
  • Badge pulse untuk status pending
  • Form input focus enhancement
Manual Enhancement
  • Tambahkan .fade-in-* untuk animasi scroll
  • Tambahkan .glass-card untuk glassmorphism
  • Tambahkan .stagger-* untuk delay
  • Tambahkan .btn-glow untuk glow
  • Tambahkan .btn-3d untuk 3D
  • Tambahkan .floating untuk icon
  • Tambahkan .counter untuk angka
